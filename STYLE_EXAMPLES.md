## Tips for Custom Styles

### Transparency Tips
- Use alpha values (0.0-1.0) for transparency
- Lower alpha = more transparent
- Good for blending with different themes

### Color Harmony
- Use complementary colors for contrast
- Use analogous colors for harmony
- Match your GTK theme for consistency

### Sizing Guidelines
- Thin bars (4-6px): Minimalist, unobtrusive
- Medium bars (7-10px): Balanced visibility
- Thick bars (11-15px): Maximum attention

### Border Radius Guide
- 0px: Sharp, technical look
- 2-4px: Subtle rounding, modern
- 5-8px: Smooth, friendly
- 9-15px: Pill-shaped, very rounded

### Color Picker Resources
- Use GNOME's built-in color picker
- Match colors from your wallpaper
- Use color theory: complementary, triadic, etc.
- Tools: `gpick`, `gcolor3` for Linux

## How to Apply

1. Open preferences: `gnome-extensions prefs calendar-progress@gnome-extensions`
2. Adjust sliders and color pickers to match desired style
3. Changes apply immediately
4. Use "Reset" button to start over if needed

## Creating Your Own Theme

1. Start with default settings
2. Adjust one property at a time
3. Preview changes in real-time
4. Take notes of values you like
5. Share your preset with others!
